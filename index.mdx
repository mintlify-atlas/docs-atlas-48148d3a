## What is FrankerFaceZ?

FrankerFaceZ (FFZ) is a browser extension and userscript that enhances Twitch with features the platform doesn't provide. It has been actively maintained since Twitch's React rewrite in 2017 and is used by hundreds of thousands of viewers and streamers.

## How FFZ works

FrankerFaceZ loads as a browser extension or userscript on Twitch. It intercepts and augments Twitch's React application by patching components through a webpack module compatibility layer, then injects its own module tree on top.

```
Twitch Page Load
      │
      ▼
  FFZ Entry Point (entry.js / entry_ext.js)
      │
      ▼
  FrankerFaceZ Core (main.ts)
      │
      ├── SettingsManager
      ├── AddonManager
      ├── TranslationManager (i18n)
      ├── SocketClient
      ├── PubSubClient
      └── Site Module (twitch-twilight)
              │
              ├── Chat Module
              │     ├── Emotes
              │     ├── Badges
              │     ├── Tokenizers
              │     └── Actions
              └── Other Site Modules
```

<Note>
  FrankerFaceZ uses a debug mode during development. Set `localStorage.ffzDebugMode = true` in your browser console on Twitch to load from a local development server.
</Note>
