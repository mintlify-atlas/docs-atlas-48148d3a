---

This guide gets you from zero to a working FFZ development environment. You'll run a local webpack dev server and load FFZ from it on Twitch.

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [pnpm](https://pnpm.io/) package manager

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/FrankerFaceZ/FrankerFaceZ.git
    cd FrankerFaceZ
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```
  </Step>

  <Step title="Start the development server">
    ```bash
    pnpm start
    ```

    This runs `webpack-dev-server` on port 8000 with hot reloading. The server self-signs a TLS certificate for HTTPS.

    <Warning>
      Your browser will show a security warning for the self-signed certificate at `https://localhost:8000`. You must accept it for FFZ to load from localhost. Visit `https://localhost:8000` directly and click through the warning.
    </Warning>
  </Step>

  <Step title="Install the FFZ userscript">
    For development, use the **userscript** version of FFZ (not the browser extension). Install a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/), then add the FFZ development userscript.

    The userscript is designed to load from your local dev server automatically when debug mode is enabled.
  </Step>

  <Step title="Enable debug mode on Twitch">
    Open Twitch in your browser and run this in the browser console:

    ```javascript
    localStorage.ffzDebugMode = true;
    ```

    Then refresh the page. FFZ will now load from your local development server at `https://localhost:8000` instead of the production CDN.
  </Step>

  <Step title="Verify the setup">
    Open your browser's developer console on Twitch. You should see FFZ initialization log messages similar to:

    ```
    [FFZ] Initialization complete in 245.12345ms.
    ```

    You can also access the FFZ instance directly:

    ```javascript
    window.ffz          // The FrankerFaceZ instance
    window.ffz.version  // Version info
    FrankerFaceZ.get()  // Same as window.ffz
    ```
  </Step>
</Steps>

## Available dev commands

| Command | Description |
|---------|-------------|
| `pnpm start` / `pnpm dev` | Webpack dev server with hot reload (development mode) |
| `pnpm dev:prod` | Webpack dev server in production mode |
| `pnpm dev:ext` | Webpack dev server for browser extension build |
| `pnpm build` | Production build |
| `pnpm build:dev` | Development build (no watch) |
| `pnpm build:ext` | Browser extension build |
| `pnpm clean` | Remove the `dist/` directory |
| `pnpm eslint` | Lint all JS and Vue files |
